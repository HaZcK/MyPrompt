Apa itu Prompt Engineering?:
Prompt engineering adalah seni dan ilmu dalam merancang instruksi (prompt) secara terstruktur agar sistem AI generatif, seperti ChatGPT atau Gemini, memahami konteks dan memberikan hasil yang paling optimal, akurat, serta relevan

apa itu user?:
user adalah individu atau pihak yang berinteraksi langsung dengan sistem kecerdasan buatan (seperti ChatGPT atau Gemini). User memberikan instruksi, mengajukan pertanyaan, atau mengunggah data untuk diproses, sementara AI bertugas merespons dan memberikan text yang sesuai

Apa itu User Input?:
Definisi: Segala bentuk perintah, pertanyaan, atau data yang Anda berikan kepada AI. User input ini sering juga disebut sebagai Prompt.
Format: Input bisa berupa teks, file dokumen, gambar, atau perintah suara.
Contoh User Input (Teks):"Tolong buatkan ringkasan materi tentang sejarah komputer dalam 3 paragraf."
Contoh User Input (File/Gambar):Mengunggah foto grafik penjualan dan meminta AI: "Jelaskan tren kenaikan pada gambar ini."
